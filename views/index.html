If the microservice gets a unix timestamp or a natural language description, it will return both.
Example usage:
https://fcc-timestam.glitch.me/December%2015,%202015
https://fcc-timestam.glitch.me/1450137600
Example output:
{ "unix": 1450137600, "natural": "December 15, 2015" }
